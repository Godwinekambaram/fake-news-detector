# 🔍 Fake News Detector AI

A Machine Learning model that detects fake news with **99.66% accuracy!**

## 🚀 What it does
Paste any news title + article text and the AI will tell you if it's **REAL or FAKE!**

## 🧠 How it works
- Dataset: 37,853 real and fake news articles
- Algorithm: Random Forest Classifier
- Accuracy: 99.66%
- Deployed as live web app using Gradio

## 📊 Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Gradio
- Google Colab

## 📁 Dataset
Download from Kaggle: [Fake and Real News Dataset](https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset)
